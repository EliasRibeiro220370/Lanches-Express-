📌 Sobre este projeto

Este repositório apresenta uma versão demonstrativa de um cardápio digital desenvolvido durante um trabalho freelance para um cliente real.

Para preservar informações comerciais e a privacidade do cliente, alguns dados, imagens e configurações foram adaptados para esta versão de portfólio.
